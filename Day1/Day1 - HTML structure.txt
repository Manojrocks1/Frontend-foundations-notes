ğŸ” Explanation of Each Part:
Tag	                        Purpose
<!DOCTYPE html>	Tells the browser this is an HTML5 document
<html lang="en">	Root element, wraps all the content. lang="en" defines the language
<head>	Contains metadata (not visible to users)
<meta charset="UTF-8">	Defines character encoding
<meta name="viewport">	Makes the site responsive
<title>	Sets the title in the browser tab
<body>	Contains all visible content (text, images, buttons, etc.)
